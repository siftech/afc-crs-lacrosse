= Fuzzing the Linux kernel with AFL

First, get the kernel into the `linux` subdirectory here. If you already have a
clone, use `git worktree` to make a worktree here. Otherwise, follow the
directions on the #link("")[forks/linux] repo.

Next, run `./configure-kernel` to generate a `.config` file for that kernel.
